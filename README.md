# LucertolaOS
Il sistema operativo ultra-leggero, gratuito e open source.

# Cos’è LucertolaOS?
LucertolaOS è un sistema operativo progettato per essere estremamente leggero e facile da usare, ideale per computer con risorse limitate. Offre un’esperienza fluida senza appesantire il sistema.

Funzionalità principali
- 🚀 Ultra-leggero e veloce
- 🔒 Codice di attivazione per sicurezza
- 💻 Versioni Home, Standard, Pro e School
- 🆓 Completamente gratuito
  
# Installazione
Segui questi semplici passaggi per installare LucertolaOS:
- Vai al sito [Crea Codici](https://demenigeco.github.io/LucertolaOS/v/act)
- Genera un codice
- Vai al [Sito Ufficiale](https://demenigeco.github.io/LucertolaOS?v=726292682927262882)
- Scegli la versione con cui hai generato il codice
- Inseriscilo nel dato campo di testo
- Clicca su 'Scarica ZIP'
